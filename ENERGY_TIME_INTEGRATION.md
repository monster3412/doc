## Интеграция энергии и времени в отчёт

### Что было добавлено

1. **Расширение функции `_enrich()`** (`src/ui.py`)
   - Теперь обрабатывает 9-11 столбцов вместо 5-ти
   - Сохраняет данные:
     - `plan_energy` (план энергии в кВт·ч)
     - `fact_energy` (факт энергии в кВт·ч)
     - `plan_time` (план времени в часах)
     - `fact_time` (факт времени в часах)
   - Агрегирует эти колонки так же, как план/факт кол-во

2. **Новые функции в `calculator.py`**
   - `calculate_energy_intensity_plan()` — плановая энергоёмкость
   - `calculate_time_intensity_plan()` — плановая трудоёмкость
   - `calculate_time_deviation()` — отклонение по времени
   - `build_efficiency_table()` — полная таблица эффективности по цехам

3. **Фрейм эффективности в UI** (`src/ui.py`)
   - Добавлена `efficiency_frame` с метриками:
     - Энергоёмкость факт/план
     - Трудоёмкость факт/план
     - Отклонение энергии и времени
     - Использование времени (%)
   - Метод `_update_efficiency_summary()` пересчитывает эти значения при формировании отчёта

4. **Обновлённый встроенный датасет** (`BUILTIN`)
   - Теперь содержит 9 столбцов вместо 5
   - Каждая запись: [workshop, date, product, plan_qty, fact_qty, plan_energy, fact_energy, plan_time, fact_time]

5. **Экспорт с энергией/временем**
   - **CSV**: дополнительные колонки для энергии и времени
   - **JSON**: дополнительные поля в объектах данных
   - **HTML**: добавлена таблица сводки эффективности с интенсивностями

### Как использовать

1. **Встроенный датасет** — при старте приложения автоматически загружается BUILTIN датасет с энергией/временем.

2. **Собственные файлы** — если загружаете CSV/Excel/JSON, убедитесь, что они содержат столбцы:
   - Для плана: `plan_qty`, `plan_energy`, `plan_time`
   - Для фактики: `fact_qty`, `fact_energy`, `fact_time`
   - (или их синонимы: `план`/`план_количество`, `факт`/`факт_количество`, и т.д.)

3. **Просмотр метрик** — сформировав отчёт, видите в UI:
   - KPI карточки (План, Факт, %)
   - **Новое:** Таблица эффективности с энергоёмкостью, трудоёмкостью и статусом цеха

### Примеры формул

```
Энергоёмкость (факт) = sum(fact_energy) / sum(fact_qty)  [кВт·ч/шт]
Трудоёмкость (факт) = sum(fact_time) / sum(fact_qty)     [ч/шт]
Энергоёмкость (план) = sum(plan_energy) / sum(plan_qty)  [кВт·ч/шт]
Трудоёмкость (план) = sum(plan_time) / sum(plan_qty)     [ч/шт]
Отклонение энергии = sum(fact_energy) - sum(plan_energy) [кВт·ч]
Отклонение времени = sum(fact_time) - sum(plan_time)     [ч]
Использование времени = (sum(plan_time) / sum(fact_time)) * 100  [%]
```
